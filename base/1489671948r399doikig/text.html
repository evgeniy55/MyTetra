Полезный инструмент для создания точек восстановления системы. Использование этого приложения позволит при необходимости в последствии восстановить систему в более ранней стадии. При этом количество точек восстановления не лимитировано. Для удобства на каждую точку можно оставлять комментарий, чтобы при удалении случайно не уничтожить нужную точку восстановления.

Процесс можно автоматизировать, настроив создание точек по расписанию - так всегда можно быть уверенным, что сохранена самая актуальная версия рабочей системы. Интерфейс утилиты достаточно прост и не вызовет каких-либо проблем с освоением даже у неопытных пользователей.
